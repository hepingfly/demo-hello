# 我的第一篇博客

欢迎来到我的个人博客！这是我的第一篇文章。

## 关于这个博客

这个博客使用 VitePress 搭建，托管在 GitHub Pages 上。我将在这里分享：

- 💻 技术学习笔记
- 🛠️ 项目实践经验
- 📚 读书心得

## 技术栈

本博客使用的技术：

- **VitePress**: 静态站点生成器
- **GitHub Pages**: 免费托管
- **GitHub Actions**: 自动化部署
